
 💸 FinanceIQ — AI Financial Intelligence Dashboard

🔗 Live Demo: https://financeiq-dashboard-six.vercel.app/

FinanceIQ is a **premium AI-powered financial dashboard** designed to transform raw financial data into actionable insights.  
It simulates a real-world fintech SaaS product with intelligent analytics, modern UI/UX, and interactive visualizations.



 🚀 Why This Project?

Most dashboards just show numbers.  
**FinanceIQ tells a financial story.**

- Detects spending patterns 🧠  
- Provides AI-like insights 📊  
- Helps users make smarter financial decisions 💡  



 ✨ Core Features

 📊 Financial Pulse (Hero Section)
- Real-time balance tracking
- Monthly performance insights
- AI-generated financial feedback
- Trend visualization (mini sparkline)

 📈 Flow Curve
- Smooth balance trend graph
- Gradient-based visualization
- Time-series financial tracking

 🧠 Smart Signals (AI Insights Engine)
- Weekend overspending detection
- Food expense increase alerts
- Savings growth tracking
- Actionable recommendations

 🗺️ Spend Map
- Category-wise expense breakdown
- Interactive donut chart
- Visual spending distribution

 📋 Activity Ledger
- Transaction history with filters
- Category-based organization
- Add transaction interaction (with toast feedback)

 🧭 Recent Activity Timeline
- Chronological event tracking
- Visual indicators for income & expenses



 🤖 AI Copilot (Highlight Feature)

An intelligent in-app assistant that enhances user experience:

- Floating chat interface
- Smart queries:
  - "Where am I overspending?"
  - "How can I save more?"
  - "Show risky patterns"
- Context-aware responses based on financial data



 🎯 Wealth Goals (Personal Finance Layer)

- Goal tracking system
- Progress visualization
- Predictive insight:
  > “At your current savings rate, you will reach this goal in X days”
- Motivational UX design



🎨 UI/UX Highlights

- 🌙 Dark-first design system
- ⚡ Smooth micro-interactions & transitions
- 💎 Premium SaaS-inspired layout
- 📱 Fully responsive design
- 🎯 Clean and scalable component architecture



 🔔 Micro-Interactions

- Toast notifications (e.g. transaction success 🚀)
- Hover animations & transitions
- Loading skeletons for better UX perception



 🛠️ Tech Stack

- Frontend: React + TypeScript  
- Styling:  Tailwind CSS  
- Build Tool: Vite  
- State Management: React Context API  
- Deployment:  Vercel
